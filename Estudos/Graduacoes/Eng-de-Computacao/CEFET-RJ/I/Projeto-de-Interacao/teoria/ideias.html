Ideias

* O que são as Ideias do Projeto de Interação?

      É uma lista das ideias feitas para dar inicio as fases de pré a pós-produção onde elas seram desenvolvidas conforme o PDF da Aula 0 em tipo de interface GUI(Graphical User Interface), PUI(Perceptual User Interface) ou WUI(Web User Interface), partindo da AHT(Analise de Tarefas) para a IHC(Interação Humano Computador).

* As principais ideias

  • Sobre Designs Comuns e de Protótipos CAD e BIM: Essa é a que ganhou.


      > Referência de nomes sobre Design CAD e BIM:
        Lumin, Star, Flamb, Ink, Sun, Cad, Eng, Space, Cink, Cinker ...


      > Simbologia:
        Engrenagem, Lampada, Sol, Pincel, Caneta, espaço, estrelas.


      > Palheta:
        Amarelo, Vermelho, Preto, Branco, Laranja, Azul.


      > O Projeto:
        Esse projeto tem como objetivo a criação de uma plataforma com um software que permite os usuários a criarem designs comuns e de prototipagem com software CAD e BIM. entre variadas comunicações audiovisuais com exclusividade a integração para o CAD(Computer Aidded Design) para criação de objetos 2D e 3D bem como a impressão desses projetos em impressoras 3D que é um recurso em ascensão desde 1984, vai ser uma platarforma voltada para aqueles que são Estudantes, Engenheiros ou Arquitetos, e Empresários para se reunirem em grupos de até 5 pessoas trabalhando de forma simultanêa excedendo os usuários podem pagar por um plano premium de R$8,50 por mês para aumentar o limite de simultaneos em incluindo até empresarial que pode chegar a 20 pessoas. O programa irá conta com integração a vários versionadores de código como o GitHub para guardar os projetos e terá uma área de marketplace que permite os usuários cadastrados na plataforma venderem seus designs ou projetos pelo preço que quiserem, a platarforma garante não cobrar nenhum valor de comissão por venda para si e ainda poderá o usuário trocar por venda usar seus pontos para resgatar GitCards no marketplace. Apresento a(o)"SpacEye".

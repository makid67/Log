    * Linguagens de Desenvolvimento e Programação FullStack *

      * Front-End: Linguagens usadas para desenvolvimento e programação com exclusividade em utilização da interface do usuário "client-side".


      * Back-End: Linguagens usadas para programação com exclusividade em promover a captura e análise de dados do "server-side" para o "client-side".


      * Fullstack: É o conjunto das linguagens "front-end" e "back-end".

* Categorias de Linguagens de Dev. e Prog.

  • Linguagens Compiladas:
      São Linguagens de Prog. / Dev. que compilam os arquivos das linhas de código num pacote que são usados em aplicações para hardwares específicios bem como o software por eles depurados, é mais rápida em resposta porém com menos portabilidade.


  • Linguagens Interpretadas:
      São Linguagens de Prog. / Dev. que compilam os arquivos de linha de código linha a linha seguindo o padrão de leitura de sintaxe de cada uma, porém são mais lentas na resposta e com uma portabilidade muito melhor.
